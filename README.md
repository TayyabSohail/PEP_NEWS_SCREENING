# PEP_NEWS_Screening
THIS IS THE PROPERTY OF AWAAZ AI (NSTP)
This is a Full-Stack development of the PEP NEWS Screening Web Application used by Bank Islami.

## TECHNOLOGIES USED
1) REACT + VITE
2) AntDesign
3) TypeScript
4) MERN Stack
